Deploy¶

In this tutorial we are going to deploy a Flask application to Heroku. Here’s what we’re going to need for this tutorial:

  • A free Heroku account.
  • Python version 2.7 installed locally - see the installation guides for MacOS, Windows, and Linux.
  • Setuptools & Pip locally. See the Python install guides above for installation instructions.
  • Virtualenv installed locally. Accomplish this by running pip install virtualenv

Why is deployment necessary?¶

Your web application must live somewhere other than your own desktop or laptop. A production environment is the version of your current application and its associated data.

Set up¶

You should go to https://www.heroku.com/ and follow the steps create a free account.

Also, you must download and install the Heroku Toolbelt from the official web-cite https://toolbelt.heroku.com/debian

Now you may use heroku in your command line.

Now you should Login to Heroku using your email address and password you used when creating Heroku account:

$heroku login
